Projekt jest zrobiony póki co przejściowo, dużo rzeczy jest zrobionych tymczasowo i nie działa. Plansza zostanie zmieniona na plansze mojego wyboru, pionki tez - UI zostanie poprawione graficznie. Gra planszowa póki co nie posiada żadnych funkcjonalności poza wykorzystaniem publicznego API RandomNumber do rzucania losoywch liczb kości.
Do zmian: -wyglad UI, -pionki, -plansza, -lekka zmiana wygladu widokow przedplanszowych. Cała funkcjonalności gry ruszania pionków itd. do zrobienia. Grafiki ogólnie zostaną poprawione tak by nie było. Kod był pisany przy użyciu Gemini/StackOverflowa/Reddita/Internetu ogólnie oraz własnej wiedzy w tym notatek z lekcji i zdjęć kodu, więc jest dużo do pozmieniania.
